<h2>🌈Installation</h2>

<b>1.Preferences > Browse Packages 

2.Create a new folder with any name 

3.Transfer all files there (without Readme.md and Images folder) 

4.Preferences > Select Color Theme 

5.Choose the theme you need </b>
